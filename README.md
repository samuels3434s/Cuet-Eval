# CUET UG Response Sheet Evaluator 🟢

A premium, privacy-first, client-side web tool to calculate CUET UG scores instantly. It parses the TCS iON response sheet HTML and matches it against copied answer key text.

Developed by **[@pxl_.dust](https://instagram.com/pxl_.dust)**.

## Features
- **Emerald Aurora Theme**: Glowing modern green and cyber cyan dark interface.
- **Sleek Logo & Favicon**: Custom generated high-tech brand assets.
- **Local file reading**: Save response sheets as HTML and upload them directly.
- **Smart Text Parsing**: Select all text from the portal answer key, copy it, and paste it. The engine automatically finds Question and Option ID pairs.
- **Detailed dashboard**: View subject-wise breakdowns, accuracy percentage, total scores, and interactive charts.
- **Exporting**: Download results as CSV or JSON instantly.

---

## How to Run Locally

Double-click the **`index.html`** file in the project directory to open it in any web browser.

## How to Host Online (Free)

You can host this website online for free so other students can use it. Since it runs completely in the client's browser, there are no database or server costs.

### Easiest: Netlify Drop (Takes 1 Minute)
1. Open **[Netlify Drop](https://app.netlify.com/drop)** in your browser.
2. Drag and drop the **`cuet-evaluator`** folder into the upload box.
3. Netlify will deploy the site instantly and give you a public link (e.g. `https://name.netlify.app`).
4. Go to site settings in Netlify to customize the URL subdomain!

### Alternative: Vercel CLI / Drag & Drop
1. Open **[Vercel](https://vercel.com/)** and sign in.
2. Add a new project and upload the folder, or use the drag-and-drop interface.
3. Vercel will launch your site at a custom `.vercel.app` domain.
